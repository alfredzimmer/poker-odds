# Poker Equity Calculator

A Texas Hold'em poker equity calculator that calculates win probabilities using Monte Carlo simulation.

## Features

- Multi-player hand comparison (2-9 players)
- Single-hand strength evaluation against random opponents
- Real-time odds calculation with visual pie chart
- Interactive card selection interface
- Dark/light mode toggle
- Automatic state persistence via localStorage

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the application.
